# 🎵 Music Desktop Application

A modern and user-friendly Music Desktop Application designed to provide a seamless music listening experience with an intuitive interface. This project focuses on creating a clean UI, smooth navigation, and an enjoyable user experience.

---

## 📖 Overview

The Music Desktop Application allows users to browse, search, and manage their favorite songs with a visually appealing desktop interface.

This project was designed in **Figma** before development to ensure a user-centered design process.

---

## ✨ Features

- 🎧 Modern Desktop UI
- 🔍 Search Songs
- ❤️ Favorite Playlist
- 📂 Music Library
- ▶️ Play / Pause Controls
- ⏭️ Next & Previous Track
- 🔊 Volume Control
- 📃 Playlist Management
- 🌙 Dark Theme Interface
- 📱 Responsive Layout (Desktop)
